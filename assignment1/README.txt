The assignment code consists of 4 python files

Main.py - triggers the code for problems
ID3.py - ID3 implementation of both binary and multi way decision tree
Tree.py - Tree Class to store decision trees
Regression.py - Linear Regression code

The code requires pandas and pandas_ml libraries to compile and run successfully

run Main.py to triggere script run for question 1, 2 and 6. The data files have to reside in the same directory.
